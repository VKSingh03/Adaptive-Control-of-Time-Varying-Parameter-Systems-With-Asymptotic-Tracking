For running the simulation 

The simulation is done in two different files, and each file represent each control method ( Proposed and E_mod). 

Steps for running the Proposed method. 
1. Open the "Proposed method Simulink model.slx" in simulink. 
2. Double click on function block to open the MATLAB code. 
3. Run the code param=[18.1502;36.2946;0.8982;1.0552]; in the command window.
4. Now run the simulink model. 
5. Check the scope for error e, Torque u, Function Estimation Error Ydtilde. 
6. Take the rms, values for e, Y, Tau for the table for the time periods as per report. 

Steps for running the e_mod method. 
1. Open the "E_Mod Simulink model.slx" in simulink. 
2. Double click on function block to open the MATLAB code. 
3. Run the code param1=[12.5;9.7877;9.7319]; in the command window.
4. Now run the simulink model. 
5. Check the scope for error e, Torque u, Function Estimation Error Ydtilde. 
6. Take the rms, values for e, Y, Tau for the table for the time periods as per report.
